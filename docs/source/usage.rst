Usage
=====

.. _installation:

Installation
------------

To use ``csst-ifs-gehong``, first install it using ``pip``:

.. code-block:: console

$ pip install csst-ifs-gehong

Execution
----------------

1-D Spectrum
~~~~~~~~~~~~~~~~

``spec1d``

Emission Line from Ionized Gas
++++++++++++++++++++++++++++++

``spec1d.IonizedGas``

Continuum of Stellar Population in Galaxies
+++++++++++++++++++++++++++++++++++++++++++

``spec1d.StellarPop``

Spectrum of Single Stellar
++++++++++++++++++++++++++

``spec1d.SingleStellar``

Spectrum of AGNs
++++++++++++++++

``spec1d.AGN``
